1. Clonar
  git clone url_del_repositorio
2. Agregar cambios
  git add -A
3. Hacer commit, guardar los cambios en el repositorio local
  git commit -m "mensaje de cambios"
4. Enviar los cambios al repositorio remoto
   git push
5. Obtener cambios del repositorio remoto
   git pull
6. Observar los cambios en el repositorio local 
   git status
